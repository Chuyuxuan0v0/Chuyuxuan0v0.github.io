js隐式转换

隐式转换定义

在js中，当运算符在运算时，如果两边数据不统一，CPU就无法计算，这时我们编译器会自动将运算符两边的数据做一个数据类型转换，转成一样的数据类型再计算。这种无需程序员手动转换，而由编译器自动转换的方式就称为隐式转换。例如1 > "0"这行代码在js中并不会报错，编译器在运算符时会先把右边的"0"转成数字0然后在比较大小。

隐式转换规则

类型 符号使用
转成string类型 +（字符串连接符）
转成number类型 ++ --(自增自减运算符) + - * / %(算术运算符) > < >= <= == != === !=== (关系运算符)
转成boolean类型 !（逻辑非运算符)
